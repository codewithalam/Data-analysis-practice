# 📊 Matplotlib Practice — Data Visualization Fundamentals

This repository documents my hands-on practice with **Matplotlib**, as part of my journey to become a data analyst.

## 🎯 What this covers

- Line, bar, and scatter charts
- Chart customization (colors, gridlines, figure sizing, legends)
- Histograms and pie charts for distribution and proportion analysis
- Subplots for multi-chart dashboards
- Plotting directly from pandas DataFrames
- Annotations and exporting charts as image files (`savefig`)

## 🛠️ Tools used

- Python
- Matplotlib
- Pandas
- Jupyter Notebook

## 📁 Files

- `matplotlib.ipynb` — practice notebook covering the topics above, using a sample ice cream sales dataset (temperature vs. sales trends)

## 📌 Notes

Each exercise includes a short written insight, not just a chart — practicing the habit of translating a visual into a one-sentence business takeaway.
